# Expense Splitter App - Backend

## Overview
The backend of the Expense Splitter App is built using **ASP.NET Core Web API**. It provides RESTful API services for **user authentication, group management, expense tracking, and debt settlement**.

## Features
✅ **JWT Authentication** – Secure user login & registration.  
✅ **Group Management** – Create groups, invite users, and view group details.  
✅ **Expense Tracking** – Add expenses, manage participants, and record payments.  
✅ **Debt Management** – View and settle outstanding balances.  
✅ **MS SQL Database** – Stores user, group, and expense data securely.  
✅ **RESTful API** – Provides endpoints for frontend integration.  

## Tech Stack
- **ASP.NET Core Web API** – Backend framework.  
- **Entity Framework Core** – ORM for database interactions.  
- **MS SQL Server** – Relational database.  
- **JWT Authentication** – Secure login and authorization.
